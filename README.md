# cs-overlay-collection

## Requirements
`node.js`

## Setup
Copy `gamestate_integration_toasthud.cfg` into `CS2_INSTALL_DIR/game/csgo/cfg`

Run `npm install express ws electron` in this directory

Run `node server.js` from commandline to start the server

Run `npm start` to start the HUD on your screen

Add a browser source in OBS with the link you see in commandline (make sure custom CSS is empty)

Enjoy
